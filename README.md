# Color Code Gnerator

## Requirements

- Python 3.11 or later

## Quick Start

```python
from colgen.utils import generate_hex_codes_from_rgb

dict_input = {
    "blue-gray": (130, 160, 170),
    "dark-blue-gray": (60, 93, 105)
}

dict_out = generate_hex_codes_from_rgb(dict_input)
```
